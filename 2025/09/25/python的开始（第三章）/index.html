布尔类型和比较运算符

布尔类型不仅可以自行定义，同时也可以通过计算得到，这里的计算即判断运算
在python中，布尔运算符有:

  • == 判断是否相等（a==b) 为true
  • != 判断内容是否不相等 （a!=b)为true

  • <判断大小，与上文相同

  • =,<=大于等于，小于等于，与上文相同

if语句的基本格式

if 要判断的条件:
条件成立时，要做的事情

elif 是else if的缩写，所以和if一样elif可以有判断条件，多条件判断是，个个条件是互斥的
if语句可以简洁输入,优势是不需要输入多余的变量
